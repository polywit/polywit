## Using a Package Manager
You can easily add this project to your own by using your favorite project manager. Here are some examples for popular build tools:

<Tabs>
	<TabItem label="pip">
		```bash
		foo@bar:~$ pip install polywit
		```
	</TabItem>
</Tabs>

For more information about the project and its usage, you can visit the PyPI page [here](https://pypi.org/project/polywit/).
